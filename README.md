#  Computer Vision Practice Repository

A structured collection of OpenCV-based Python programs to help beginners learn and practice core Computer Vision & Image Processing concepts step-by-step.

This repository focuses on hands-on learning with simple, well-explained scripts.

---

## 📌 Features

-  Video Processing (read, display, save, trackbars)
-  Image Drawing (shapes like circle, line, rectangle, arrow)
-  Text Annotation on images
-  Image Transformations (rotation, resizing basics)
-  Image Display comparison: OpenCV vs Matplotlib
-  Grayscale & Color image handling
-  Interactive controls using Trackbars

---

## 📂 Project Structure

###  Video Processing
- `reading_video.py` → Read and display video using OpenCV  
- `video_saving.py` → Save video from webcam or file  
- `trackbar.py` → Control image/video properties using trackbars  

###  Image Drawing & Shapes
- `circle.py` → Draw circles on images  
- `line.py` → Draw straight lines  
- `arrowedline.py` → Draw arrowed lines  
- `rectangle.py` → Draw rectangles  

###  Text & Annotation
- `puttext.py` → Add text on images using OpenCV  

###  Image Transformations
- `rotate.py` → Rotate images using OpenCV  

###  Image Reading & Display Methods
- `sample.py` → Load and display image  
- `read_cv_and_show_plt.py` → OpenCV vs Matplotlib comparison  
- `read_show_grayimg_opencv.py` → Display grayscale image  
- `read_show_coloring_mat.py` → Show colored image using Matplotlib  
- `read_mat_show_opencv.py` → Convert Matplotlib image to OpenCV format  

---
